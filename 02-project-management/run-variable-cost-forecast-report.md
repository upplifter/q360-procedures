# Run the Variable Cost Forecast Report

## What This Procedure Does

This procedure shows you how to run the Variable Cost Forecast Report. The report projects variable costs for future periods, including material and other variable expenses, helping you forecast project expenses and manage budgets.

## Before You Begin

Ensure you have access to the Projects module and Reports section. You should understand which projects and cost categories are relevant to your analysis. Have your forecast period in mind before starting.

## Steps

### Step 1: Navigate to the Report

From the Main Menu, click **Projects**. In the Projects menu, click **Reports**. Select **Variable Cost Forecast** from the report list.

### Step 2: Set Filter Criteria

The report filter panel opens with fields for customizing your forecast. Set the filters using the table below:

| Field | What to enter |
|-------|---------------|
| Project | Select specific projects or leave blank for all |
| Forecast Period | Choose the period you want to forecast (next quarter, next year, etc.) |
| Cost Category | Select variable costs, material costs, or all cost types |
| Division | Choose a division or leave blank for all |
| Department | Select a department or leave blank for all |

Click **Apply Filters** to update the report with your selections.

### Step 3: Run the Report

Click the **Run** or **Refresh** button to execute the report with your chosen filters.

### Step 4: Review the Report Output

The report displays forecasted variable costs broken down by project, period, and cost category. Review the totals to understand projected spending. Compare forecast amounts to your budget to identify potential overages or savings.

### Step 5: Export or Print (Optional)

To save the report, click **Export** and choose your format (Excel, PDF, or CSV). To print, click **Print** and follow your printer settings.

## What Happens Next

The forecast data displays current projections based on your selected parameters. Use this information for budget planning, cost control, and financial forecasting. Share the report with project managers and finance teams for review and decision-making.

## Common Issues

**Forecast amounts seem too high or too low**
Verify that the correct forecast period and cost categories are selected in your filters. Forecasts are based on historical data and rates, so ensure you are reviewing the appropriate time frame. Adjust filters and run again if needed.

**Missing project data in the forecast**
Check that the projects you need are active and have variable cost information recorded. Projects without cost history may not appear in the forecast. Confirm your filter selections include the correct projects.

**Negative forecast values appearing**
Negative values may indicate credit memos or cost adjustments in historical data. Review these entries to determine if they represent legitimate credits or data entry errors. Contact the project manager if values appear incorrect.

**Report runs slowly with many projects selected**
Limit the number of projects in your filter or select a shorter forecast period to improve performance. Run the report again after adjusting your selections.

**Cannot modify filters or they do not save**
Refresh the report screen and try adjusting filters again. If filters still do not respond, clear your browser cache or close and reopen the report. Contact your system administrator if the issue continues.
