# Run the Service Contract Profitability Report

## What This Procedure Does

The Service Contract Profitability Report shows revenue and costs across service calls, orders, and direct postings for each contract. Use this report to evaluate contract financial performance, identify unprofitable contracts, and optimize pricing and resource allocation.

## Before You Begin

- Have permission to access Service or Accounting Reports
- Determine the date range for analysis
- Identify contract or customer filters needed
- Prepare to review detailed contract margins

## Steps

### Step 1: Open Service Contract Profitability Report

Go to Service > Reports > Service Contract Profitability (or Accounting > Reports). The Print Report dialog opens with filter options.

### Step 2: Set Filter Criteria

Configure filters to define the analysis:

| Field | What to select |
|-------|----------------|
| Company | Select the company |
| Branch | Select the branch (optional) |
| Department | Select the department (optional) |
| Date Range | Enter start and end dates |
| Customer | Select customer (optional) |

### Step 3: Run the Report

Click Run Report. The report generates and displays in Print Report dialog or Live Data grid.

### Step 4: Review Contract Profitability Metrics

The report displays per contract:

- Contract No
- Customer Name
- Contract Type
- Contract Revenue (from calls, orders, and direct postings)
- Material Costs
- Labor Costs
- Subcontractor Costs
- Other Costs
- Total Cost
- Gross Profit
- Profit Margin (%)
- Profitability Status (color-coded: green profitable, red unprofitable)

Summary rows aggregate totals by customer, branch, or department.

### Step 5: Analyze High and Low Performers

Identify profitable contracts worth protecting and growing. Highlight unprofitable contracts requiring price adjustments, cost reductions, or termination. Compare margins across similar contract types and customer segments.

## What Happens Next

Results inform pricing decisions, contract negotiations, and resource allocation. Use findings to justify price increases, identify cost reduction opportunities, or plan contract terminations. Re-run the report quarterly to track profitability trends.

## Common Issues

**Contract Shows Negative Profit**
Review all costs posted to the contract. Verify time bills and parts are correctly priced. Check if discounts are appropriate given the service level. Consider renegotiating contract terms or pricing.

**Totals Do Not Match General Ledger**
Verify the date range matches your accounting period. Check that all cost categories are included. Confirm unposted entries are properly reflected. Consult with accounting team on reconciliation.

**Cannot Identify Specific Cost Driver**
Use the detailed view or zoom function to see line-by-line costs by call or order. Drill down to understand which cost categories impact profitability most.
