# Run the Technician Performance Summary Report

## What This Procedure Does

The Technician Performance Summary Report aggregates KPIs for all technicians including total call counts, average response times, and average call duration. Use this report to evaluate technician productivity, identify high performers, and plan training or incentives.

## Before You Begin

- Have permission to access Service Reports
- Determine the date range for evaluation
- Identify performance standards and targets
- Understand technician roles and responsibilities

## Steps

### Step 1: Open Technician Performance Summary Report

Go to Service > Reports > Technician Performance Summary. The Print Report dialog opens with filter options.

### Step 2: Set Filter Criteria

Configure filters to define the analysis:

| Field | What to select |
|-------|----------------|
| Company | Select the company |
| Branch | Select the branch (optional) |
| Department | Select the department (optional) |
| Date Range | Enter start and end dates |

### Step 3: Run the Report

Click Run Report. The report generates and displays in Print Report dialog or Live Data grid.

### Step 4: Review Technician KPIs

The report displays aggregated metrics per technician:

- Technician Name
- Total Calls
- Average Response Time (from dispatch to arrival)
- Average Call Duration (time on site)
- First Call Fix Rate (if available)
- Customer Satisfaction Rating (if available)
- Total Hours
- Revenue Generated (if available)

### Step 5: Analyze and Compare Performance

Identify top performers with high call counts and short response times. Compare technicians in the same specialty or region. Identify those needing additional training or coaching. Recognize high performers for incentive or promotion purposes.

## What Happens Next

Results support performance reviews, compensation planning, and workforce development. High-performing technicians can be recognized and retained. Lower performers may need coaching or skill development. Use the report to set individual targets and track improvement over time.

## Common Issues

**Missing Technicians in Results**
Verify technicians have calls assigned in the system during the date range. Check that no technicians are on leave or inactive during the period. Confirm technician records are properly maintained.

**Metrics Show Inconsistencies**
Verify the date range is accurate. Check that all calls are properly assigned and timed. Review individual call records if specific metrics seem incorrect.

**Cannot Drill Into Individual Calls**
Use the Technician Performance Detail Report (SVC-RPT-011) to see line-item call data for a specific technician. Filter by technician in the detail report for deeper analysis.
