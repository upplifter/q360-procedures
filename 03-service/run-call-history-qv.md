# Run the Call History Quick View

## What This Procedure Does

The Call History Quick View provides a searchable log of all service call records with key details. Use this report to find specific calls, view customer service history, or analyze call patterns over time.

## Before You Begin

- Have permission to access Service Quick Views
- Identify the time period and search criteria
- Know the customer, product, or call type you're searching for

## Steps

### Step 1: Open Call History Quick View

Go to Service > Quick Views > Call History. The Quick View opens with filter options on the left panel.

### Step 2: Set Filter Criteria

Configure filters to search for calls. Available filters include:

| Field | What to select |
|-------|----------------|
| Company | Select the company |
| Branch | Select the branch (optional) |
| Department | Select the department (optional) |
| Date Range | Enter start and end dates |
| Status | Select call status (Open, Closed, etc.) |
| Call Type | Select call type to filter |
| Customer | Select customer (optional) |
| Product | Select product (optional) |

Click Run or Refresh to apply filters.

### Step 3: Review Call List in Grid

The grid displays each call with key details:

- Call No
- Customer
- Call Type
- Status
- Date Created
- Date Closed
- Product
- Issue Summary
- Technician assigned

Sort results by clicking any column header. Use text search at the top of the grid to find specific calls by keyword.

### Step 4: Drill Into Call Details

Click any call number in the grid to zoom into the full service call record. Review all details including problem description, solution, time bills, parts, charges, and customer notes.

### Step 5: Use Results for Analysis

Identify patterns in the call history such as recurring issues, frequently called products, or high-activity customers. Export the grid to Excel if you need to perform additional analysis.

## What Happens Next

The Quick View displays all calls matching your filter criteria. Results update as new calls are created and existing calls are modified. Use call history for customer research before new calls, dispute resolution, warranty analysis, and trend identification.

## Common Issues

**Cannot Find Specific Call**
Expand the date range to ensure the call falls within the search period. Try searching by customer name or call type instead of exact call number. Verify the call has not been deleted or archived.

**Grid Displays Too Many Results**
Add more specific filters such as Status, Call Type, or Customer to narrow results. Use the date range to limit to recent calls. Consider searching in a shorter time window.

**Drill-Through to Call Detail Not Working**
Verify the call is not deleted or archived. Check user permissions to access the call record. Ensure the call number is properly linked in the system.
