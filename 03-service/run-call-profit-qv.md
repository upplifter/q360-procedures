# Run the Call Profit Quick View

## What This Procedure Does

This Quick View displays financial profitability data for service calls including revenue, material costs, labor costs, subcontractor costs, and gross profit margins. Use this report to analyze call profitability and identify high-cost or unprofitable calls.

## Before You Begin

- Service calls must have time bills and parts entries completed
- Have permission to access Service Quick Views
- Determine the time period and filters you want to apply

## Steps

### Step 1: Open Call Profit Quick View

Go to Service > Quick Views > Call Profit. The Quick View opens with filter options on the left panel.

### Step 2: Set Filter Criteria

Configure the filters to narrow results. Common filters include:

| Field | What to select |
|-------|----------------|
| Company | Select the company to report on |
| Branch | Select the branch (optional) |
| Department | Select the department (optional) |
| Date Range | Enter the start and end dates |
| Call Type | Select call type to filter (optional) |

Click Run or Refresh to apply the filters.

### Step 3: Review Results in Grid

The grid displays each service call with the following columns:

- Call No
- Customer
- Revenue
- Material Cost
- Labor Cost
- Subcontractor Cost
- Misc Cost
- Total Cost
- Gross Profit
- Gross Margin (%)

Review the results to identify calls with high revenue, high costs, or low margins. Sort by any column by clicking the column header.

### Step 4: Drill Into Call Details

Click any row in the grid to zoom into the full service call record. Review detailed time bills, parts, and charges. This helps understand which line items contributed most to the call's financial performance.

## What Happens Next

The Quick View displays current financial data for all calls matching your filter criteria. Changes to time bills or parts are reflected in the next refresh. Use these results to monitor call profitability, optimize pricing, and identify operational improvements.

## Common Issues

**No Results Displayed**
Verify the date range includes calls with completed entries. Ensure the company and branch filters match your data. Check that time bills and parts have been entered and posted for the calls.

**Totals Do Not Match Expected Values**
Confirm all time bills and parts for the calls have been posted. Unposted entries do not appear in financial reports. Check that discounts or special adjustments are correctly applied.

**Cannot Click Through to Call Detail**
Verify the call number in the grid is active and not deleted. Check user permissions to access the full call record.
